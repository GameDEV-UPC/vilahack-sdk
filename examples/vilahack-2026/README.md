# Vilahack 2026 examples.

This folder contains a couple of examples to integrate vilahack-sdk to the frontend projects using the architecture stack of the Vilahack 2026.

To use the examples you need to have acces to Vilahack and Supabases API's. Then copy .env-template to .env and fill variables.

> [!WARNING]
> Be carefull and don't push any private keys to the repo.

## Examples

If you wanna try the examples install the dev dependencies with npm install and then run the scripts with tsx.

- fetch-setup.ts: Simple example to set up auths and endpoints to able to fetch from the Vilahack API.
